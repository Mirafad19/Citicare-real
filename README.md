# Citicare Integrated Health Solutions

A modern, highly optimized, patient-centered digital healthcare platform committed to bridging the gap between individuals and professional, high-quality healthcare services.

## Overview

Citicare Integrated Health Solutions offers exceptional client-centered home nursing, specialty care, baby nursing, geriatric care, and specialist consultations. This web application serves as the primary gateway for users to book appointments, manage their healthcare needs, and learn about the extensive range of professional medical care services provided by experienced, verified professionals.

## Key Features

- **Nursing Services**: A customizable booking engine for specialized on-site, baby, senior, cosmetic surgery, hospice, and 24-hour nursing care.
- **Specialist Care**: Digital discovery and consulting scheduling across major healthcare departments.
- **Responsive Navigation**: Smooth mobile and desktop drop-down systems for seamless user flow.
- **Enterprise Design**: Engineered with high-contrast, professional design pairings using Inter and deep-indigo/slate palettes.

## Local Development

### Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Configure Environment Variables**:
   Create a `.env.local` file in the root directory and add any required configurations shown in the `.env.example` file.

3. **Start the Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

4. **Build for Production**:
   ```bash
   npm run build
   ```

## Technology Stack

- **Framework**: [React 19](https://react.dev/) / [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: [Motion](https://motion.dev/)
